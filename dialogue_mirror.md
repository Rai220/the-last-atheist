# Зеркальные диалоги — ключевой приём

Главная ирония игры: Алексей оказывается в **абсолютно симметричной ситуации** с верующими, которых высмеивал всю жизнь. Эта таблица — шпаргалка для написания диалогов.

## Таблица зеркал

| Что говорили верующие на Земле | Что говорит Алексей в аду |
|---|---|
| «Я просто чувствую присутствие Бога» | «Я просто чувствую, что это симуляция» |
| «Посмотри вокруг — мир слишком совершенен для случайности» | «Посмотри вокруг — ад слишком абсурден для реальности» |
| «У тебя нет доказательств, что Бога НЕТ» | «У тебя нет доказательств, что это НЕ симуляция» |
| «Ты просто не хочешь видеть правду» | «Вы просто не хотите видеть паттерны» |
| «Однажды ты поймёшь» | «Однажды вы все поймёте» |
| «Наука не может объяснить всё» | «Ваша "реальность" не может объяснить NPC-поведение демонов» |
| «Миллионы людей не могут ошибаться» | «Миллионы строк кода не могут быть случайностью» |
| «Это требует веры, а не доказательств» | «Это требует анализа, а не слепого принятия» |
| «Бог работает загадочными путями» | «Код работает загадочными путями — потому что legacy» |
| «Ты слишком гордый, чтобы принять истину» | «Вы слишком вшиты в систему, чтобы увидеть матрицу» |

## Где использовать

- **Judgment (дебаты с Богом)** — Бог может цитировать Алексея ему же обратно
- **Hell (разговоры с душами)** — другие души ловят его на противоречии
- **Matrix route** — момент осознания: он слышит свои слова из чужих уст
- **Breakdown** — кульминация: Алексей сам замечает симметрию

## Правило

Не указывать на зеркало напрямую. Игрок должен заметить сам. Единственное исключение — финальная реплика Бога:

> «Ты прав. Или не прав. Это не имеет значения. Потому что теперь ты знаешь, каково это — верить без доказательств. И теперь ты знаешь, почему они верили.»

## Отсылка к «Контакту» Сагана

Элли Эрроуэй — учёный, пережила контакт, не может доказать. Палмер Джосс — богослов, требует от неё доказательств. Роли меняются.

Алексей = Элли. Бог = Палмер. Симуляция = контакт. Доказательств нет.
