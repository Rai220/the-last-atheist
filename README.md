# Последний Атеист

**Визуальная новелла о программисте-атеисте, который умирает и попадает на Страшный Суд. Вместо того чтобы раскаяться, он решает, что это симуляция. Потому что ну серьёзно.**

> *«38 лет. Смерть на пробежке. Первой за полгода. F.»*

---

## О чём это

Алексей Волков — IT-шник, рационалист, завсегдатай r/DebateReligion. Умирает. Оказывается в загробном мире с очередями, бюрократией и формой 66-А.

Ему предлагают принять реальность. Он предлагает найти баг в системе.

- **~30 концовок** — от организации профсоюза в аду до романа с демоницей
- **Зеркальная философия** — атеист без доказательств верит в симуляцию, как верующие верили в Бога
- **Чёрный юмор** — HR ада, пасхалки на Бесконечное Лето, DDLC, Зайчик, и Невзоров
- **Система статов** — argument_quality, wtf_level, lilith_interest влияют на развитие сюжета

## Скриншоты

Стиль: Disco Elysium meets божественная бюрократия.

## Как запустить

### Браузер (рекомендуется)

```bash
# Установить зависимости
bun install

# Запустить dev-сервер с live-reload
bun run serve
```

Откроется на **http://localhost:5100**

### Веб-билд

```bash
bun run build:web
```

Результат в `./build/web` — статический сайт, можно залить на любой хостинг.

### Desktop (Electron)

```bash
yarn start          # Dev
yarn build:mac      # macOS билд
yarn build:windows  # Windows билд
yarn build:linux    # Linux билд
```

### Android

APK уже собран: `the-last-atheist.apk` (Capacitor).

## Структура проекта

```
js/
  scripts/
    prologue.js    — пролог: утро, Reddit, офис, смерть (3 варианта)
    judgment.js    — Страшный Суд: очередь, дебаты с Богом
    hell.js        — ад: бюрократия, бар, бунт, Лилит, Виктор
    endings.js     — 28+ концовок
  characters.js    — определения персонажей
  assets.js        — карта ассетов (сцены, музыка)
  effects.js       — визуальные эффекты (тряска, глитч, сердцебиение)
  storage.js       — переменные и статы
  main.js          — точка входа + карта маршрутов
engine/core/       — Monogatari (движок визуальных новелл)
assets/
  characters/      — спрайты персонажей (AI-generated)
  scenes/          — фоны локаций (AI-generated)
  music/           — музыка (procedurally generated)
style/main.css     — кастомные стили
generate_art.py    — генерация арта через Google Gemini / Imagen 4.0
generate_music.py  — генерация музыки
```

## Основные маршруты

- **Вера** — принять, поставить по Паскалю, или изучать как учёный
- **Бунт** — профсоюз, хакинг бюрократии, демократия в аду
- **Бар** — открыть бар, сеть баров, или стать терапевтом душ
- **Матрица** — симуляция? спидран? медитация? dev commentary?
- **Лилит** — роман, побег, предательство
- **Секретные** — пророк, реинкарнация, нигилист, Сизиф

## Предупреждение

**18+**: Темы смерти, экзистенциального кризиса, религиозной сатиры, чёрного юмора. Если вы ищете подтверждение или опровержение существования Бога — это не сюда. Игра не даёт ответов. Она задаёт вопросы.

## Генерация ассетов

Арт и музыка генерируются через AI:

```bash
export GEMINI_API_KEY='your-key'
python3 generate_art.py          # Imagen 4.0 (сцены) + Gemini 2.5 Flash (персонажи)
python3 generate_music.py        # Процедурная музыка
python3 generate_scenes_pillow.py # Фоны через Pillow (без API)
```
